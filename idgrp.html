Besoins Fonctionnels

La plateforme dédiée aux cabinets dentaires et aux laboratoires de prothèses répond aux besoins suivants :

Pour le Cabinet Dentaire

  • Gestion de Commande :
    • Suivi et gestion des commandes pour les équipements dentaires et les consommables.
  • Gestion de la Salle d'Attente :
    • Système de gestion en temps réel de la file d'attente des patients avec affichage du temps d'attente estimé.
  • Gestion de RDV :
    • Planification et gestion des rendez-vous avec notifications automatiques par e-mail ou SMS.
  • Gestion des Patients :
    • Base de données des patients avec historique médical et de traitement.
  • Base de Données en Ligne :
    • Stockage sécurisé des dossiers patients dans une base de données en ligne accessible aux praticiens.
  • Messagerie :
    • Système de messagerie intégré pour la communication entre les professionnels de santé et les patients.
  • Notifications :
    • Notifications pour les rappels de rendez-vous, les paiements et la gestion administrative.

Pour le Laboratoire de Prothèse Dentaire

  • Gestion de Commande :
    • Suivi et gestion des commandes pour les prothèses dentaires depuis le cabinet dentaire.
  • Gestion des Tâches :
    • Organisation et suivi des tâches pour la fabrication et le traitement des prothèses.
  • Facturation :
    • Génération et gestion des factures pour les cabinets dentaires et les patients.
  • Suivi de Livraison :
    • Module pour suivre les livraisons des prothèses vers les cabinets dentaires avec mise à jour en temps réel.
  • Gestion des Techniciens :
    • Suivi des techniciens en charge des différentes commandes et tâches du laboratoire.
  • Gestion des Produits :
    • Base de données des matériaux et produits utilisés pour la fabrication des prothèses.
  • Messagerie :
    • Système de communication interne entre le laboratoire et les cabinets dentaires.
  • Notifications :
    • Notifications pour les livraisons, les paiements, et les mises à jour sur les commandes.

Module de Livraison

  • Consommation d'API de Livraison :
    • Intégration avec une API de service de livraison pour suivre l'expédition des commandes en temps réel.
    • Affichage des informations de localisation des véhicules de livraison et estimation du temps d'arrivée.

Technologies Utilisées

Pour le développement de cette plateforme, les technologies suivantes ont été utilisées :

  • Symfony : Serveur pour la gestion des requêtes backend et des APIs.
  • JavaScript : Serveur pour la gestion des requêtes backend et des APIs.
  • MySQL : Base de données relationnelle pour la gestion des patients, commandes et tâches.
  • API REST : Intégration avec des services tiers comme les API de livraison.
  • UML : Modélisation des flux de travail pour les fonctionnalités de la plateforme.
