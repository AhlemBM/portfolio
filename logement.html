Besoins Fonctionnels

La plateforme dédiée à l'agence de location de logement répond aux besoins suivants :

Pour l'Administrateur (Dashboard)

  • Gestion des Logements :
    • Ajouter, supprimer ou modifier les informations des logements disponibles à la location (photos, descriptions, prix, disponibilité).
  • Gestion des Réservations :
    • Consulter et gérer les réservations effectuées par les clients.
  • Gestion des Clients :
    • Suivi des utilisateurs enregistrés et de leurs réservations passées et en cours.
  • Statistiques :
    • Accès à un tableau de bord avec des statistiques (taux de réservation, revenus générés, disponibilité des logements).
  • Messagerie :
    • Communication avec les clients via un système de messagerie intégré pour répondre aux questions et gérer les requêtes.
  • Notifications :
    • Réception de notifications pour les nouvelles réservations, les annulations ou les demandes spéciales.

Pour les Clients

  • Consultation des Logements Disponibles :
    • Parcourir les logements disponibles avec filtres par lieu, prix, date de disponibilité, nombre de chambres, etc.
  • Réservation de Logement :
    • Effectuer des réservations en ligne en sélectionnant les dates et les logements disponibles.
  • Gestion des Réservations :
    • Consulter l’historique des réservations, modifier ou annuler une réservation à venir.
  • Compte Utilisateur :
    • Création et gestion d'un compte utilisateur pour suivre les réservations, sauvegarder des logements favoris, et gérer ses informations personnelles.
  • Messagerie :
    • Communiquer avec l'agence via un système de messagerie intégré pour poser des questions sur un logement ou une réservation.
  • Notifications :
    • Réception de notifications pour les confirmations de réservation, les rappels de paiement et les modifications de réservation.

Technologies Utilisées

Pour le développement de cette plateforme, les technologies suivantes ont été utilisées :

  • Node.js : Serveur pour la gestion des requêtes backend et des APIs.
  • Angular : Développement du front-end pour une interface utilisateur dynamique côté client et administrateur.
  • MySQL : Base de données relationnelle pour la gestion des logements, clients, réservations et transactions.
  • API REST : Pour la communication entre le front-end et le back-end, ainsi que l'intégration de services tiers (paiements, messagerie, etc.).
  • UML : Modélisation des flux de travail pour les fonctionnalités de la plateforme.
